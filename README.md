# Advent of Code 2024 — Day 2: Red-Nosed Reports

Solution to Day 2 of Advent of Code 2024, written in JavaScript (Node.js).

## Usage

Add your puzzle input as `input.txt` in the same directory, then run:
```bash
node day2.js
```

## Problem

Count how many sequences are "safe": all increasing or all decreasing, with each adjacent pair differing by 1 to 3.
